# 🐍 Python For Loops for Beginners

A **for loop** is used to repeat a block of code a specific number of times. Think of it like an automation machine that goes through a list of items one by one.

---

## 💻 Code and Examples

Here is how you write and use a for loop in Python:

```python
# --- 1. BASIC SYNTAX ---
# 'sequence' can be a list, a range of numbers, or text.
for item in sequence:
    # Code to execute goes here
    pass


# --- 2. LOOPING THROUGH A LIST ---
# This goes through a list and prints every item.
fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

# Output:
# apple
# banana
# cherry


# --- 3. LOOPING THROUGH A RANGE OF NUMBERS ---
# The range(5) function counts from 0 to 4.
for i in range(5):
    print(i)

# Output:
# 0
# 1
# 2
# 3
# 4
```

---

## ⚠️ Key Rules to Remember

* **🔍 Indentation:** The code inside the loop *must* be spaced to the right (usually 4 spaces).
* **Do not forget the `:`** Always put a colon at the end of the `for` statement line.
* **📛 Variable Name:** You can name the temporary variable (`fruit` or `i`) whatever you want.
